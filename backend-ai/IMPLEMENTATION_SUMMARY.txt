================================================================================
SIMILARITY LOGIC
================================================================================

Model Distance Calculation:
  - Peugeot 206 vs 208: distance = 2  → similarity = 0.333
  - Peugeot 207 vs 208: distance = 1  → similarity = 0.500
  - Peugeot 3008 vs 208: distance = 2800 → similarity = 0.000

Formula: similarity_score = 1 / (1 + distance)

Requirements:
  ✓ Same make required (Peugeot ≠ Renault)
  ✓ Same part_category required (phare = phare)
  ✓ Different model required (exact match handled earlier)
  ✓ Valid price > 0
  ✓ Not expired
